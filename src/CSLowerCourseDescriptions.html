CS1030: Foundations of Computer Science

Prerequisites: None

Instructor: Chad Mano

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: BPS

Investigation of computers and computing in today’s society, including the basic scientific and mathematical concepts that underlie computer science, computing, and computer systems.

 

CS1400: Indroduction to Computer Science - CS 1

Prerequisites:

One of the following:

  • MATH 1050 with a grade of C- or better
  • Math ACT score of 25 or higher or equivalent SAT score
  • AP Calculus AB score of 3 or higher

Languages, Frameworks, and Tools:

  • Python

Instructors: Chad Mano, Dan Watson (Fall), Nicholas Flann (Spring), Erik Falor (Summer) (Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 4

Course Attributes: None

Introduction to science of problem solving, programming, program development, algorithm analysis, and data structures. Students will learn to develop correct software in a current programming language environment.

CS1440: Methods in computer Science

Prerequisites:

  • CS 1400 with a grade of C- or better

Languages, Frameworks, and Tools:

  • Python
  • Git
  • Shell (GitBash)

Instructors: Erik Falor
(Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: None

CS 1440 helps students become “good” software developers using the basic programming concepts acquired CS 1400. This class is not focused on learning a features or constructs of a programming language; rather, it teaches principles and practices that help manage complexity, improve software quality, and leverage existing or new technologies.

CS2410: Introduction to Event Driven Programming and GUIs

Prerequisites:

  • CS 1410 with a grade of C- or better

Languages, Frameworks, and Tools:

  • Java
  • Android SDK (Software Development Kit)

Instructors: Joseph Ditton
(Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: None

Students learn the fundamentals of event-driven software, common graphical user interface (GUI) components, GUI development environments and a new programming language. They also strengthen their understanding of essential object-oriented concepts, including classes, inheritance, and polymorphism.

 

CS2420: Algorithms and Data Structures - CS3

Prerequisites:

Both of the following:

  • GPA of 2.0 or better
  • CS 1410 OR ECE1410 (With proof of Java compentency approved by the Computer Science Department)

Languages, Frameworks, and Tools:

  • Java

Instructors: Vicki Allan, Haito Wang (Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: QI

Students learn various data structures and algorithms, and refine their programming skills. Course topics include strategies for the design and evaluation of algorithms, algorithm analysis, sorting, trees, graphs, hash tables, priority queues, disjoint sets, dynamic programming, and greedy algorithms.

 

CS2610: Develping Dynamic, Database-Driven, Web Applications

Prerequisites:

  • CS 2410 with a grade of C- or better

Languages, Frameworks, and Tools:

  • Javascript
  • HTML
  • CSS
  • Python
  • Django
  • Json

Instructors: Erik Falor (Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: None

Develop secure, dynamic, database-driven web applications. Learn the fundamentals of building web pages. Add interactive capabilities with client-side and server-side technologies. Learn about information representation, storage, retrieval and transmission in Internet-based environments.

 

CS2810: Computer Systems Organization and Architecture

Prerequisites:

Both of the following:

  • CS 1410 with a grade of C- or better
  • 2.0 GPA

Languages, Frameworks, and Tools:

  • Python

Instructors: Heng-Da Cheng, Daniel Watson (Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: None

This course examines the organization and architecture of computer systems. It covers terminology, data representation, Boolean Algebra, and combinational and sequential logic circuits as they apply to computer hardware and software.

 

3000 Level Courses

CS3100: Operating Systems and Concurrency

Prerequisites:

Both of the following:

  • CS 2420 with a grade of C- or better
  • 2.0 GPA
  • Admission to Professional Program

Languages, Frameworks, and Tools:

  • N/A

Instructors: Chad Mano, Mahdi Nasrullah Al-Ameen (Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: None

Design and implementation of operating systems. UNIX will be used as one example, but all categories of operating systems will be discussed. Presentation of the concept of concurrency as it applies to operating system design and application.

 

CS3200: Mobile Application Development

Prerequisites:

Both of the following:

  • CS 2410
  • CS 2610 with a grade of C- or better
  • 2.0 GPA
  • Admission to Professional Program

Languages, Frameworks, and Tools:

  • Java
  • Adroid SDK

    • OR

  • Swift
  • Mac

Instructors: Joseph Ditton (Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: None

This course is an exploration of mobile application development. Students will design, build, and potentially publish several Android or iOS applications over the course of the semester. Topics include a high-level overview of mobile platforms, history of mobile devices, mobile economics, ethical advertising, Firebase, maps, GPS, device to device communication, mobile sensors, advanced UI, animation, cross-platform development, and publishing mobile applications.

 

CS3430: Scientific Computing with Python

Prerequisites:

Both of the following:

  • CS 1410
  • MATH 1210 or MATH 2270 with a grade of C- or better
  • Admission to Professional Program

Languages, Frameworks, and Tools:

  • Python

Instructors: Vladimir Kulyukin (Subject to change)

Regularly offered in Spring semesters

Credits: 3

Course Attributes: DSC, QI

Scientific Computing (SciComp) is the collection of algorithms, tools, and theories required to solve numerical problems in science, engineering, mathematics, economics, etc. By the end of this course, students will have mastered or become familiar with such topics as linear systems, LU decomposition, linear programming, simplex, differentiation engines, Newton-Raphson algorithm, central divided difference, integration approximation, periodic function approximation with Fourier series, congruence systems, Richardson's extrapolation, Romberg integration, decision trees, Huffman codes.

 

CS3450: Introduction to Software Engineering

Prerequisites:

Both of the following:

  • CS 2420 with a grade of C- or better
  • CS 2610 with a grade of C- or better
  • Admission to Professional Program

Languages, Frameworks, and Tools:

  • N/A

Instructors: Daniel Watson, Mario Harper (Subject to change)

Regularly offered in Fall and Spring semesters

Credits: 3

Course Attributes: CI

Students learn the fundamentals of software engineering principles and practices, including project management, configuration management, requirements definition, system analysis, design, testing, and deployment. This course provides students with hands-on experience in a group software development project.

 

CS3460: Modern C++

Prerequisites:

Both of the following:

  • CS 1440 with a grade of C- or better
  • CS 2420 with a grade of C- or better
  • Admission to Professional Program

Languages, Frameworks, and Tools:

  • C++
  • CMake

Instructors: James Mathias (Subject to change)

Regularly offered in Fall semesters

Credits: 3

Course Attributes: None

This course is intended to develop the student’s understanding and use of the C++ programming language, with a particular focus on the latest C++ standards. It focuses on C++ as a systems language and emphasizes application/computational performance. Topics include history of C++, C++ ecosystem (e.g., CMake, gtest), dynamic memory, object oriented C++, functional C++, generic programming, and miscellaneous subjects such as containers, date and time handling, random numbers, and algorithms.
